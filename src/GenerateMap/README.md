# GenerateMap 将地图保存至图片

- 作者: 少司命, Cai
- 出处: 无
- 生成地图图片

## 指令

| 语法        |     权限      |   说明   |
|-----------|:-----------:|:------:|
| /map img  | generatemap | 生成地图图片 |
| /map file | generatemap | 生成地图文件 |

## REST API

| 路径                |     权限      |      说明      |
|-------------------|:-----------:|:------------:|
| /generatemap/img  | generatemap | 获取地图图片base64 |
| /generatemap/file | generatemap | 获取地图文件base64 |
